Guia de Configuração 3 etapas

Siga os passos abaixo uma única vez. Após concluído, o cadastro e o mapa funcionarão automaticamente.

Etapa 1 — Criar a planilha no Google Sheets

Passo 1

Acesse sheets.google.com e crie uma nova planilha em branco.

Passo 2

Renomeie a aba (guia inferior) para exatamente: Cadastros

Passo 3

Na primeira linha, crie os cabeçalhos nas células A1 a E1:

A1: Nome    B1: País    C1: Estado    D1: Cidade    E1: Data

Passo 4

Copie o ID da planilha da barra de endereço do navegador. É o trecho longo entre /d/ e /edit:

https://docs.google.com/spreadsheets/d/SEU_ID_AQUI/edit

Guarde esse ID — você vai precisar dele no arquivo mapa.html.

Etapa 2 — Criar o Apps Script (recebe os cadastros)

Passo 1

Na planilha, clique no menu Extensões → Apps Script.

Passo 2

Apague o código existente e cole o código abaixo no editor:

function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var sheet = SpreadsheetApp.getActiveSpreadsheet()
                              .getSheetByName('Cadastros');
    sheet.appendRow([
      data.nome  || '',
      data.pais  || '',
      data.estado|| '',
      data.cidade|| '',
      data.data  || new Date().toLocaleDateString('pt-BR')
    ]);
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'ok' }))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({ status: 'erro', msg: err.toString() }))
      .setMimeType(ContentService.MimeType.JSON);
  }
}

function doGet(e) {
  return ContentService
    .createTextOutput('Apps Script ativo.')
    .setMimeType(ContentService.MimeType.TEXT);
}

Passo 3

Clique em Salvar projeto (ícone de disquete ou Ctrl+S).

Passo 4 — Publicar como Web App

Clique em Implantar → Nova implantação.

Em "Tipo", selecione App da Web.

Configure:

Executar como:        Eu mesmo (sua conta Google)
Quem tem acesso:      Qualquer pessoa (anônimo)

Clique em Implantar e autorize as permissões quando solicitado.

Passo 5

Copie a URL da implantação (começa com https://script.google.com/macros/s/...).

⚠️ Guarde essa URL — ela não aparece novamente sem reimplantar.

Etapa 3 — Configurar os arquivos HTML

No arquivo cadastro.html

Localize a linha:

const APPS_SCRIPT_URL = 'COLE_AQUI_A_URL_DO_SEU_APPS_SCRIPT';

Substitua pelo valor que você copiou no Passo 5 acima.

No arquivo mapa.html

Localize a linha:

const SHEET_ID = 'COLE_AQUI_O_ID_DA_PLANILHA';

Substitua pelo ID da planilha copiado na Etapa 1, Passo 4.

Tornar a planilha pública (leitura)

Na planilha, clique em Compartilhar → Alterar para qualquer pessoa com o link, com permissão de Visualizador. Isso permite que o mapa leia os dados sem login.

✓ Apenas leitura — ninguém pode editar a planilha pelo link público.

Como hospedar e compartilhar

Opção recomendada — GitHub Pages (gratuito)

1. Crie uma conta em github.com (gratuito).

2. Crie um repositório público, faça upload dos dois arquivos (cadastro.html e mapa.html).

3. Em Settings → Pages, ative o GitHub Pages na branch main.

4. Você receberá URLs como:

https://seu-usuario.github.io/grupo-estudos/cadastro.html
https://seu-usuario.github.io/grupo-estudos/mapa.html

Outras opções gratuitas

Netlify — arraste os arquivos e publique em segundos.
Vercel — similar ao Netlify, muito simples.

Resumo final

✓ Envie o link do cadastro.html para os membros do grupo.
✓ Visualize o mapa em mapa.html a qualquer momento — ele lê a planilha em tempo real.
✓ Pontos maiores = mais membros na mesma cidade. Passe o mouse para ver a cidade.
